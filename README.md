# Data-Analytics-Portfolio
A multivariate data analysis and interactive Tableau dashboard exploring simulated plant communication

___Project Overview
This project explores the fascinating concept of plant communication through measurable biological signals. The goal is to classify the type of message a simulated plant is likely transmitting (Warning, Contentment, Distress, or Invitation) based on environmental factors and biological outputs.

___Methodology
To investigate the relationship between environmental factors and plant development, I constructed a scatter plot correlating sunlight exposure hours with growth rates, incorporating a color encoding for plant message types to enable a multivariate analysis. 

Subsequently, I integrated this scatter plot with an aggregated bar chart into a unified, interactive dashboard. To enhance analytical depth, I configured the bar chart to function as a dynamic filter. This interactivity allows the user to isolate specific message categories (such as Warning or Contentment) and observe how the growth-sunlight correlation shifts across different biological states, ultimately providing a more granular understanding of the simulated plant communication data.
